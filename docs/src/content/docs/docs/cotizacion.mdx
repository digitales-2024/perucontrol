## Aprobar cotización

Cuando el cliente acepta la cotización:

<Uml code={`
@startuml
|Cotizacion|
start
if (Existe cotizacion pendiente*?) then (si)
else (no)
	:Crear cotización;
	end
endif

:Selecciona cotizacion;
:Marca como aprobado;


if (Crear proyecto automaticamente?) then (si)
	|Proyecto|
	:Crear proyecto;
	end
else (no)
endif
|Cotizacion|

end
@enduml
`} />
